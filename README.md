# selenium-examples
Учебные примеры работы с Selenium.
## Список файлов:
- find_attribute_by_picture.py - чтение значения из атрибута.
- sum2atrributes.py - сумма 2 атрибутов.
- checkbox_and_radiobutton.py - обратная капча (работа с флажками и радиокнопками).
- xpath_form.py - заполнение формы с помощью XPATH.
- alert_process.py - работа с JS-сообщением.
- tab_switch.py - смена вкладки в браузере программным путём.
- no_such_element.py - попытка обратиться к несуществующему элементу.
- implicit_wait.py - работа с неяным ожиданием (пока не выполнится условие)
